Working With GIT

Before creating a directory in GIT, please make sure to create GIT account or login.

Steps to create a GIT repository:

  1. Install GIT HUB desktop/ GIT BASH applications.

  2. Login using GIT account. Click the image to view the screenshot.
  3. Create a new local repository using "New Repository" option in File Menu. Click the image to view the screenshot.

  4. Create two folders in the local repo named "Source" and "Documentation". Click the image to view the screenshot.

  5. Source folder is used to store the code files.

  6. Documentation folder is used to store the documentation and image files.
